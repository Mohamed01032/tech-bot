# TechBot — Professional AI Support Agent

TechBot is a high-end technical support chatbot built with **Flask** and powered by **Anthropic's Claude AI**. It features a modern, responsive interface with professional design aesthetics.

## ✨ Features

- 🚀 **Advanced AI Integration**: Powered by Claude-3-Sonnet for expert technical advice.
- 🎨 **Premium UI/UX**: Modern Glassmorphism design with a sleek dark theme.
- 📝 **Markdown Support**: Beautifully formatted responses with syntax highlighting for code.
- 📱 **Fully Responsive**: Works seamlessly on Desktop and Mobile.
- 🔒 **Security First**: Environment-based API key management.
- 🛠️ **Utility Tools**: Copy code blocks with one click, clear chat history, and auto-expanding input.

## 🛠️ Installation

1. **Clone the project** and navigate to the directory.
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Configure your API Key**:
   - Rename `.env.example` to `.env`.
   - Add your Anthropic API Key: `ANTHROPIC_API_KEY=your_key_here`.
4. **Run the application**:
   ```bash
   python app.py
   ```
5. Open your browser at `http://127.0.0.1:5000`.

## 📂 Project Structure

- `app.py`: Backend Flask server & AI logic.
- `static/`: Contains CSS and JS for the premium frontend.
- `templates/`: HTML structure.
- `.env`: Secure configuration.
